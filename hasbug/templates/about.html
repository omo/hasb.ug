What is hasb.ug?

Hasb.ug is a URL shortener for bug tracker URLs.

For example, hasb.ug shortens a long Bugzilla URL https://bugs.webkit.org/show_bug.cgi?id=63606 to http://wkb.ug/63606.

You can register your own domain or host name (like "wkb.ug") to hasb.ug server and associate specific URL pattern to the name. Hasb.ug redirects the short version to associated long version. This name-pattern pair is called a shortener. Sign in using your Github account to tell hasb.ug about your shorteners.

Hasb.ug also provides names under hasb.ug domain for you. For example, you can create less.hasb.ug as a short version of https://github.com/cloudhead/less.js/issues/. You don't need to pay for any domain names if you use names under hasb.ug domain.

Domain Setup

Once purchased a domain for your bug, you need to

  • Point 50.19.127.155 as your domain's A record. This is the IP address of hasb.ug server. Then,
  • Tell the domain to hasb.ug through the console page.

Note: For hasb.ug sub-domains, what you need to do is just to tell it from the cosole. No DNS configuration is needed.
