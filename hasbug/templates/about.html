What is hasb.ug?

Hasb.ug is a URL shortener for bug tracker URLs. For example, hasb.ug shortens a long Bugzilla URL https://bugs.webkit.org/show_bug.cgi?id=63606 to http://wkb.ug/63606.

You can register your own host name (like "wkb.ug") to hasb.ug server and associate specific URL pattern to that host name. Then hasb.ug redirect the short version to the associated long version. You need to sign in using your Github account to tell hasb.ug about your shortener.

Hasb.ug also provides its sub-domains for you, which is *.hasb.ug. For example, you can create less.hasb.ug as a short version of https://github.com/cloudhead/less.js/issues/. You don't need to own your own domain if you use hasb.ug sub-domains.

Domain Setup

Once purchased a domain for your bug, you need to

  • Point 50.19.127.155 as your domain's A record. This is the IP address of hasb.ug server and
  • Pell the domain to hasb.ug through the console page.

Note: For hasb.ug sub-domaains, what you need to do is just to tell it from the cosole. No DNS configuration is needed.
